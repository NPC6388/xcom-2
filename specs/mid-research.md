# Mid × Research

**Phase:** Mid (first Skulljack → Avatar Project reveal) | **Section:** Research | **Leads with:** Timing/tempo
**Difficulty:** Commander | **Scope:** base + integrated DLC, no WOTC
Prereqs per [tech-tree.md](tech-tree.md).

> Tempo thesis: mid game is a **race to Elerium**. Elerium gates every top tier (plasma, powered
> armor), and it needs **Plated Armor + Gauss (Mag Cannon)** first. In parallel, start the **Shadow
> Chamber / Codex Brain** chain so the Avatar clock doesn't outrun you. Slot key autopsies for the
> upgrades they unlock.

## Notes
- **Beat (entering mid):** you've just skulljacked an Officer → a Codex appeared. Get **Codex Brain** going and keep pushing the Gauss→Elerium line.
- **Autopsy discipline:** treat autopsies as *targeted* picks for a specific unlock or threat, not a checklist — every detour delays Elerium.
- **Psi decision point:** committing to Psi means Sectoid Autopsy → Psionics → Psi Lab → train operatives. It's strong but it's a real time/space investment; decide here, not late.

### Locked ✓ (from parsed tech tree)
- **Elerium ← Gauss Weapons + Plated Armor** — confirmed. Gauss Weapons sits strictly before Elerium.
- Skulljack Officer → Kill a Codex → **Codex Brain** → (Encrypted Codex Data → Skulljack Codex → Neutralize Avatar) and (Codex Brain Coordinates → Psionic Gate). See [tech-tree.md](tech-tree.md).
