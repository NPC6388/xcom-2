▍ Early Game

Gatecrasher → first Skulljack / first Black Site
Tempo goal: Be 6 soldiers strong in Magnetic Weapons + Predator Armor before you Skulljack an ADVENT Officer.

The highest-value early "upgrade" isn't gear — it's the 5th squad slot. Then cheap weapon mods and a Proving Ground experimental grenade.

Ability picks — A-team leveling

  • Ranger: Phantom → Shadowstrike/Conceal (scouting); Bladestorm situational.
  • Sniper: Long Watch, Lightning Hands, Quick Draw; Serial later.
  • Grenadier: Shred (mandatory), Demolition, Holotargeting.

Your two Specialists

① HACKER (offensive Gremlin)
Combat Protocol → Haywire Protocol (seize turrets/MECs/Sectopods) → Capacitor Discharge → Guardian. Pump the Hacking stat so Haywire lands on hard robots.
② MEDIC (support)
Medical Protocol → Revival Protocol → Field Medic → Restoration (AoE heal). Your sustain backbone.
🔧 Raise the hacker's Hacking: feed them the skulljack (Skullmining = permanent +Hacking/+Intel) · equip a Hacking PCS · battlefield hacks roll bonuses. (method source blocked; verify)
➡️ Transition to MID when: you build the Skulljack and skulljack an ADVENT Officer — spawns a Codex and reveals the path toward the Avatar Project. (Verify Codex-spawn.)

▍ Mid Game

Skulljack → Codex → Avatar Project reveal
Tempo goal: Win the race to Elerium (needs Plated Armor + Gauss) while starting the Shadow Chamber / Codex Brain chain. Whole squad on mag/Gauss + Bluescreen before robot-heavy pods scale.

Avatar facilities → final mission

Maxed Hacking + Haywire Protocol seizes enemy MECs/Sectopods/turrets — one of the strongest tempo swings in the finale. Keep the Hacking PCS on them.

Core spine (parsed from the darosh SVG — locked ✓)

  • Weapons: Modular → Magnetic → Gauss Weapons → [Elerium] → Plasma Rifle → Beam Cannon / Plasma Lance / Storm Gun
  • Armor: Hybrid Materials → Plated Armor (→ Predator, EXO, Spider) → [Elerium] → Powered Armor (→ Warden, W.A.R., Wraith)
  • THE GATE — Elerium ← Gauss Weapons + Plated Armor. Unlocks both top tiers.
  • Biotech → every autopsy + AWC. Officer Autopsy → Proving Grounds, MEC Breakdown (→ Bluescreen Rounds), Sectopod Breakdown
  • Sectoid Autopsy → Mindshield → Psionics → Psi Lab
  • Avatar spine: Proving Grounds → Skulljack → Skulljack Officer → Kill a Codex → Codex Brain → (Neutralize Avatar) + (Psionic Gate) + (Recovered Stasis Suit) → Avatar Autopsy → Broadcast Tower → Final Mission
ℹ️ Every autopsy needs the unit's corpse + Biotech researched. Full edge list (159 nodes / 178 edges) in specs/tech-tree.md; regenerate via python parse_tree.py.
